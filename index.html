Les langages du Web

Séance 1

Introduction

Les langages HTML et CSS sont les principaux langages du Web.

L'importance des deux langages est schématisée dans les deux photos précédentes :

  • Le HTMLest utilisé pour construire la structure (le squelette) d'une page Web.
  • Le CSS est utilisé pour embellir et donner une apparence attractive de la page Web.

Les deux langages sont complémentaires.

Structure d'une page Web

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  
</body>
</html>

Explication

<!DOCTYPE html>
Indique au navigateur que la page utilise la version 5, selon W3C, du langage HTML
<html lang="en">
...
</html>
Indique le début et la fin du document HTML, ainsi que la langue utilisée dans ce document, utile pour les screen readers.
<head>
...
</head>

Marque l'entête du document HTML. C'est la partie invisible de la page Web qui contient des instructions destinées au navigateur pour l'aider à bien afficher la page.

On y trouve par exemple :

<meta charset="UTF-8">
Indique l'encodage des caractères, l'UTF-8 est un encodage universel qui permet d'afficher toutes les langues (Latin, Arabe, Chinois, Japonais, etc.). 

<meta name="viewport" 
content="width=device-width, initial-scale=1.0">
Indique que la page peut-être rendue sur un mobile sans subir d'agrandissements, ni de rétricissements. 

<title>Document</title>
Indique le texte qui s'affiche dans la barre du titre du navigateur. 

<body>
  ...
</body>
Tout le contenu qui sera affiché dans le navigateur doit être écrit à l'intérieur de cette balise.

Les premières balises

La page est composée de plusieurs éléments visibles :

  • Une liste non ordonnée
  • Un titre et un sous-titre
  • Des paragraphes

Liste non ordonnée

Pour construire la liste ci-dessus on peut écrire le code suivant:

<ul>
  <li>What is it?</li>
  <li>What is used for?</li>
  <li>What types?</li>
  <li>What is composed of?</li>
</ul>

Les titres

Le langage HTML supporte 6 niveaux pour les titres, le titre principal est indiqué par un titre niveau 1 <h1>.

Les titres montrés dans l'image ci-dessus sont écrites en HTML :

<h1>Computers</h1>
<h2>What is a computer?</h2>

Les paragraphes

La balise <p>...</p> délimite les paragaphes dans un document HTML. Anisi, les paragraphes ci-dessus peuvent être écrites en HTML de la façon suivante :

<p>A computer is a <b>machine</b> that can be instructed to carry out sequences of arithmetic or logical
  operations automatically via <i>computer programming</i>. Modern computers have the ability to follow
  generalized sets of
  operations, called <b>programs</b>. These programs enable computers to perform an extremely wide range
  of tasks.</p>
<p>A "complete" computer including the hardware, the operating system (main software), and peripheral equipment
  required and used for "full" operation can be referred to as a <b>computer system</b>. This term may
  as well be used for a group of computers that are connected and work together, in particular a <u>computer
    network</u> or <u>computer cluster</u>.</p>

Mise en forme

On remarque qu'il y a des mots soulignés, en gras et en italique.

Nous utilisons la balise :

  • <u>souligné</u> pour souligner un texte.
  • <b>gras</b> pour mettre un texte en gras.
  • <i>Italique</i> pour mettre un texte en italique.
